Overview

This repository contains a collection of SQL assignments completed on two relational database systems: Company_SD and BikeStores.
The scripts demonstrate strong SQL fundamentals and intermediate–advanced querying skills through real business scenarios including HR management, sales analytics, product data, and customer behavior.

The 5 SQL files cover the following domains:

Data Manipulation (INSERT, UPDATE, DELETE)

Data Retrieval (SELECT queries)

Joins (Inner, Outer, Self, Multi-table)

Subqueries (Single-row, Multi-row, ANY, ALL, IN)

Aggregation & Grouping

String Functions, Date Functions & Data Cleaning

Sales Performance Calculations

Set Operations (UNION, INTERSECT, EXCEPT)

Window-style pagination (TOP, OFFSET … FETCH)

Business Logic Queries (Managers, Supervisors, Store performance)
